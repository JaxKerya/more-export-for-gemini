Export your Gemini Deep Research reports in any format you need.

"More Export for Gemini" extends Google Gemini's built-in share menu with powerful, 100% offline export options.

DOCUMENTS
- Word (.docx) - Real OOXML document, opens in Microsoft Word, Google Docs, LibreOffice
- PDF (.pdf) - Uses your browser's native print-to-PDF for perfect output
- EPUB (.epub) - Valid EPUB 3 e-book with a table of contents and rendered math
- LaTeX (.tex) - Ready-to-compile source for academic writing
- RTF (.rtf) - Rich text for any word processor
- HTML (.html) - Standalone styled webpage, ready to open in any browser
- Reader HTML - A clean, distraction-free reading view with theme and layout options

TEXT & DATA
- Markdown (.md) - GitHub, Obsidian, Notion and CommonMark flavors
- Plain Text (.txt) - Clean, readable text without any markup
- JSON (.json) - Structured data for developers and automation tools
- CSV (.csv) - Extracted tables as spreadsheet-ready data (opens correctly in Excel)

CITATIONS & BIBLIOGRAPHY
- BibTeX (.bib), RIS (.ris), CSL-JSON (.json) - Import straight into Zotero, Mendeley, EndNote
- 9 citation styles: APA, MLA, IEEE, Chicago, Harvard, Vancouver and more
- Automatic DOI / ISBN detection from source links (offline)

BUNDLES & CLIPBOARD
- Vault bundle (.zip) - Obsidian-ready notes with separated tables and references
- Download All (.zip) - Every enabled format bundled in a single ZIP
- Copy as Markdown, plain text, rich HTML (pastes formatted into Google Docs) or JSON

KEY FEATURES
- Export profiles: save complete setups (formats + options) and switch them with one click
- Export history: recent reports are backed up locally and can be re-exported offline - even after the Gemini conversation is gone
- Page & typography controls: paper size, margins, font size, line spacing and font family for PDF, HTML, Word and LaTeX
- Source hygiene: de-duplicate, sort and enrich your reference list
- Faithful math (KaTeX), code highlighting, tables, footnotes and right-to-left language support
- Interface in 8 languages: English, Turkish, Spanish, Portuguese (Brazil), German, French, Japanese, Korean - switchable in settings
- Keyboard shortcuts for your primary format, Copy as Markdown and Download All
- Seamless integration into Gemini's native menu, plus a right-click context menu

PRIVACY
This extension does not collect, store, or transmit any data. All processing happens locally in your browser. No analytics, no tracking, no external servers.

HOW TO USE
1. Open a Gemini Deep Research report
2. Click "Share and export" - new options appear at the bottom
   (or right-click anywhere and choose "More Export for Gemini")
3. Pick your format and download!

Works only on gemini.google.com Deep Research pages.
